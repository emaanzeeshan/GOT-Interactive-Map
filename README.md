# The History of Westeros

An immersive, responsive web experience exploring the history of Westeros and Essos through an animated interactive map, timeline, house profiles, dragons, battles, and characters.

## Features
- Cinematic loading experience
- Interactive map with location details
- Draggable historical timeline
- Great Houses, characters, dragons, battles, and castles sections
- Searchable archive
- Audio toggle and ambient theme sounds

## Run locally
Open index.html in a browser, or serve the folder with any static server.
